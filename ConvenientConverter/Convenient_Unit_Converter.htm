Tool Instructions
This application allows you to create a 'sheet' of unit conversion sets that can be saved for later use. Multiple such customized sheets can be created. Several sets of defaults unit conversions have been provided in different unit sections, such as Length Conversions, Mass Conversions, etc.

Unit Section:
In addition to the default unit conversions in each unit section, users can add their own conversion set that follows a linear relationship. For example, to add a relationship, A = 3B + 2, put A, 3, B, and 2 in the 4 input boxes at the bottom of the unit section and click on 'click to add to list' button. This user-defined conversion gets stored in the system and will be available upon subsequent usage of the application in the same browser.

Users can also choose a conversion combination using the drop-down lists provided in the top part of a unit section. The system will be able to provide the conversion facility for the shosen combination of units, if this unit combination can be reached via the unit combinations available in conversion sets shown in the middle part of the unit section. For example, choose to convert mile into cm and this is allowed because the following unit conversions are available: mile to ft, ft to meters, meters to inches, inches to cms, thus relating mile to cm.

To add a conversion to the 'Conversions Sheet' section, click on the Up arrow button.

Conversion Sheet:
The Conversion Sheet shows all the unit conversion that were uploaded from the different unit sections. A cross button is provided to remove any unit conversion from the Conversion Sheet. The different unit conversion boxes are sortable, i.e., they can be dragged and dropped to changes their positions within the Conversion Sheet section.

To save a Conversion sheet, click on the Save Sheet option on the navigation bar at the left. This saved sheet can be loaded back into the application be clicking the Load Sheet in the navigation bar.
